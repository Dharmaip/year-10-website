Computer

Definiton

A computer is like a smart machine that helps you do all sorts of things with information.
It can store data, process it really fast, and help you play games, do homework, or even chat

But what really makes computers special is their ability to follow instructions, called programs
or software. These instructions tell the computer what to do, how to do it, and when to do it. This
software can be anything from a simple game to sophisticated business applications.

Component

A computer has essential parts called components. Some key ones are the central processing unit (CPU),
which thinks and calculates; memory (RAM), which stores data temporarily; storage, like a hard drive, for
long-term data; motherboard, connecting everything; and input/output devices, such as keyboard and monitor,
for communication. Each part plays a crucial role!
